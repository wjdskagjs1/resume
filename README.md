# 자기소개
안녕하세요, 정남헌입니다.
저는 학교에서 C, C#, JAVA, 자료구조, 운영체제 등의 기초지식을 공부했습니다. 그 외에는 unity, android studio, flash를 다룰 수 있습니다. html, css, javascript, php&mysql, ajax를 이용한 웹 제작도 경험해본 적이 있습니다.
